# Image Assets

## Hero background

Replace `hero-scene.svg` with your custom pixel-art illustration:

- **File:** `hero-scene.webp` or `hero-scene.svg`
- **Size:** ~1920×1080, optimized for web
- **Style:** Full-bleed park/cityscape like cofounder.co

Update the `src` in `src/components/Hero.tsx` if you change the filename.

## Optional foreground

- `hero-laptop.webp` — if you prefer a photo/SVG laptop prop instead of the CSS mockup in `HeroForeground.tsx`

## Legacy placeholders

The old project/award placeholder images can be removed or kept for Phase 2 sections.
